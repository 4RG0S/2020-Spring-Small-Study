이 날은 파이썬 과제를 했습니다.
과제는 총 3가지로, 

1. 리스트 내에 있는 원소들을 특정 문자열로 치환하는 함수를 구현.
2. 문자열 내에서 특정 문자가 얼마나 사용되었는지 출력하는 함수를 구현.
3. 리스트를 입력받고, 리스트의 원소를 한 줄에 10개씩만 출력하는 함수를 구현.
